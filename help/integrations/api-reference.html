API Reference

What this covers

REST API endpoint reference for Tessallite. All endpoints require HTTP Basic authentication unless stated otherwise. See API Authentication for details on credentials and roles.

Base URL: http://HOST:3000/api/v1

REST API endpoint paths are subject to change between major versions. Check the release notes when upgrading across major versions.

Health

Method Path Auth required Description
GET /health No Returns service health status. Response: {"status":"ok","db_migration":"N"}. Use this to verify the Frontend service is reachable.

Workspaces

These endpoints require a System Admin account.

Method Path Auth required Description
GET /workspaces System Admin List all workspaces.
POST /workspaces System Admin Create a workspace. Request body: {"slug":"acme","display_name":"Acme Corp","admin_email":"admin@acme.com"}
GET /workspaces/{slug} System Admin Get details for a specific workspace.
DELETE /workspaces/{slug} System Admin Delete a workspace. This operation is irreversible and removes all projects, models, and aggregates in the workspace.

Projects

These endpoints require a Tenant Admin or Modeller account.

Method Path Auth required Description
GET /workspaces/{slug}/projects Tenant Admin or Modeller List all projects in the workspace.
POST /workspaces/{slug}/projects Tenant Admin Create a new project in the workspace.
GET /workspaces/{slug}/projects/{id} Tenant Admin or Modeller Get details for a specific project.
DELETE /workspaces/{slug}/projects/{id} Tenant Admin Delete a project and all its models and aggregates.

Aggregates and diagnostics

These endpoints require a Modeller account or higher.

Method Path Auth required Description
GET /workspaces/{slug}/projects/{id}/aggregates Modeller List all aggregates for the project, including their current status (Ready, Building, Stale, Error).
POST /workspaces/{slug}/projects/{id}/aggregates/{agg_id}/refresh Modeller Trigger an immediate refresh of the specified aggregate. The build runs asynchronously; poll the GET aggregates endpoint to check status.
GET /workspaces/{slug}/projects/{id}/diagnostics Modeller Get model health diagnostics, including schema drift warnings, permission errors, and build failure details.

Model definition

Method Path Auth required Description
GET /workspaces/{slug}/projects/{id}/model Modeller Get the model definition for a project: tables, joins, dimensions, and measures.
