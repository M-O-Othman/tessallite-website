Screenshot of the User dialog (in either System Admin or Tenant Admin) with the Role dropdown set to "modeler". Below the dropdown, a small "Effective access" matrix shows four rows — System Configuration (No access), Tenant Configuration (No access), Project Configuration (Edit (assigned)), Model Configuration (Edit (assigned)). Each row has the screen name + a one-line hint on the left and a colored chip on the right (grey for No access, green for Edit, blue for Edit (assigned)). The matrix updates immediately when the role changes. Dimensions: 460x520px.
