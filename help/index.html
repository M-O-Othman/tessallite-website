Tessallite Help

This library documents Tessallite: what it is, how to connect BI tools, how to build data models, and how to administer the platform. It covers all four user roles and all supported data sources.

Who this library is for

Analyst

Connects BI tools to Tessallite via JDBC or XMLA and queries data. Start with What is Tessallite and Connect a BI tool.

Modeller

Builds and maintains data models — defining dimensions, measures, joins, and aggregate policies. Start with Projects and models.

Tenant Admin

Manages users, roles, and settings for one workspace. Start with Workspaces and tenants and Roles and permissions.

System Admin

Deploys and operates the Tessallite platform. Start with Install locally or Install on GCP.
