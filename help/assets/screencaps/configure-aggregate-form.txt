Screenshot of the Model Builder Aggregate Drawer in create mode, opened from the Aggregates panel's New button. Right-anchored 720 px drawer with three tabs in the header — Definition (active, underlined), Schedule, Advanced. The Definition tab body shows: a Target dropdown labelled "Target (where the aggregate is stored)" set to "public"; an info hint card reading "An aggregate is a pre-computed roll-up of the model at a chosen grain (dimensions) over selected measures. The Query Router redirects matching aggregate queries to it instead of scanning the source."; a Grain (dimensions) multi-select picker; a Measures multi-select picker; an "Include quantile columns (p25, p50, p75, p95)" toggle (off). Footer carries Cancel (left) and Save (right, disabled until selections made). Behind the drawer the Canvas shows the model's dimension tables. Toolbelt visible on the left with the Aggregates icon highlighted. Header summary bar shows model name "modely", the Model Enabled switch, and Canvas / Model Health tabs. Dimensions: 1280x800px.
