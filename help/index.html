Tessallite Help

This library documents Tessallite: what it is, how to connect BI tools, how to build models, how the conversational agent works, and how to administer the platform. It covers the full product surface, from the Workspace Explorer and Model Builder to system configuration, BI integrations, and troubleshooting.

Who this library is for

Analyst

Connects BI tools to Tessallite and queries deployed models. Start with What is Tessallite, Workspace Explorer, and Connect a BI tool.

Modeller

Builds and maintains semantic models: sources, joins, measures, dimensions, calendars, aggregates, security, lineage, and diagnostics. Start with Projects and models.

Tenant Admin

Manages users, roles, SSO, webhooks, audit views, project settings, and agent configuration for one workspace. Start with Manage users.

System Admin

Deploys and operates the Tessallite platform. Start with Architecture overview, Deploy locally, or System configuration.
