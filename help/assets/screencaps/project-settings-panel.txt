Screenshot of the Admin → Projects tab with one project selected. On the right side, after the Models and Access sections, a new "Settings" section appears in its own bordered Paper. Inside it, a search box and accordion sections (Source database, Aggregate target, General) each contain rows. Each row has the dotted key + inherited-value caption on the left, a text input showing the current override (or empty with the inherited value as placeholder) in the middle, and Save / Clear buttons on the right. The "Clear" button is only visible on rows that have an explicit override; clicking it removes the override and falls back to the tenant value. Dimensions: 1280x900px.
