Screenshot of the Model Builder Settings panel with the "Configuration" sub-tab selected (third tab after General and AI). The panel shows accordion sections (Aggregates, AI scheduler, Optimizer) each holding rows of model-level setting overrides. Each row has the dotted key + inherited-value caption on the left, a text input showing the current override or the inherited placeholder in the middle, and Save / Clear buttons on the right. A small Alert banner above the rows reminds the user that model settings override project defaults and that empty input clears the override. Dimensions: 480x800px.
